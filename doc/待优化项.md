# AI PPT Agent — 待优化项清单

> 记录时间：2026-04-03
> 状态：待处理，按优先级排序

---

## P0 — 严重问题（影响基本可用性）

### 1. SVG 内容溢出画布
- **现象**：部分页面的文字/元素跑出 SVG viewBox 边界外
- **原因**：LLM 生成的坐标计算不准确，未严格遵守 x+width ≤ 1260 约束
- **方向**：Prompt 3 已加入红线约束（本次更新），需持续观察；若仍溢出可加后端校验层，自动裁剪/修正越界元素

### 2. 搜索引擎关键词提取
- **现象**：充实阶段搜索返回的信息与主题偏差大，内容过时（如搜到 Gemini 2.5 的旧信息）
- **原因**：直接把整段标题+要点拼接作为搜索词，Google 返回不精准
- **方向**：在搜索前增加一步"LLM 提取 2-3 个精准搜索关键词"，并加上时间过滤（Serper 支持 `tbs: "qdr:y"` 限定最近一年）

## P1 — 设计质量问题（影响产品竞争力）

### 3. 设计稿页面内容空洞
- **现象**：部分页面卡片里只有一两句话，大面积留白
- **原因**：充实环节的内容精炼后信息密度不够；或内容被过度截断
- **方向**：调整充实 Prompt，要求每个要点附带 1 个具体数据/案例；SVG Prompt 中允许每张卡片正文最多 5 行

### 4. 封面和目录页缺乏引导性
- **现象**：封面只有标题+副标题，缺少品牌氛围；目录页像纯文字列表
- **方向**：封面 Prompt 加入"装饰性渐变色块+几何图形+品牌色条"要求；目录页加入"章节编号+色条+简短摘要"

### 5. 缺少章节分隔页
- **现象**：PPT 中间全是 content 页，没有"Part 1"/"Part 2"这样的过渡页
- **方向**：大纲生成时在每个 part 的第一页前自动插入一张 "section_divider" 类型页面（只有章节标题+大号编号），SVG 渲染时按全屏大字设计

### 6. SVG 元素结构混乱/重叠
- **现象**：文字盖住文字，图形盖住图形，坐标计算互相冲突
- **方向**：Prompt 3 已加入严格约束（2026-04-03 更新）；长期可考虑后端增加 SVG 后处理校验层（lxml 解析后检测 bbox 重叠并自动调整）

### 7. visual hint 泄露到设计稿
- **现象**："水平流程图，节点用圆形图标串联箭头" 等描述性文字直接显示在 SVG 画面上
- **方向**：Prompt 3 已在红线中明确禁止（2026-04-03 更新）；同时在传给 SVG Prompt 的内容中过滤掉 visual 字段

## P2 — 功能增强（提升产品价值）

### 8. AI 生图接入（背景图/配图）
- **目标**：SVG 设计稿生成后，为封面/章节页自动生成背景图或配图
- **方案**：
  - 免费方案：Pexels/Unsplash 图库 API，按标题关键词搜索高质量摄影图
  - 付费方案：DALL-E / Flux / Stable Diffusion 生成定制化背景
  - 落地方式：SVG 中使用 `<image href="...">` 嵌入图片 URL
- **优先级**：SVG 设计稿质量稳定后再接入

### 9. 设计稿上直接编辑
- **目标**：用户在设计稿（SVG 视图）上点击某个文字/元素，可以直接修改内容或告知 AI 如何调整
- **方案**：
  - 前端解析 SVG DOM，识别可编辑的 `<text>` 和 `<foreignObject>` 元素
  - 点击后弹出编辑框（手动修改）或 AI 助手面板（指令式修改）
  - 修改后局部重新生成该元素的 SVG 片段，而非整页重渲
- **依赖**：需要 SVG 元素有稳定的 id 命名规则

### 10. 搜索引擎升级
- **目标**：获取更精准、更时效的搜索结果
- **方案对比**：
  | 方案 | 成本 | 精准度 | 时效性 |
  |------|------|--------|--------|
  | 当前 Serper（优化关键词） | 免费额度内 | 中→高 | 好 |
  | Tavily（AI 原生搜索） | 付费 | 最高 | 最好 |
  | Perplexity API | 付费 | 高 | 好 |
- **最小改动**：先实现"LLM 提取关键词 → 搜索"，不换引擎

## P3 — 体验优化（锦上添花）

### 11. 策划稿元素拖动
- 前端 `<motion.div>` 添加 `drag` 属性，拖拽结束更新坐标

### 12. 主题色应用到设计稿
- 当前设计稿 SVG 的配色由 LLM 自由发挥，未严格遵循 Design Token
- 可在 SVG 后处理中用正则替换颜色值

### 13. PPTX 导出优化
- 当前导出的 SVG 在 Office 中的兼容性待验证
- foreignObject 在 Office 中不支持，导出前需转换为 `<text>` + 手动换行

### 14. 多语言支持
- Prompt 根据用户语言切换（中/英）
- 搜索关键词也需要对应语言

## 技术债务

### 15. .env 与 .env.example 不同步
- 每次改 .env 后 .env.example 没更新，容易混淆

### 16. 热重载不可靠
- uvicorn --reload 经常不加载新的 .py 文件
- 需要手动清 __pycache__ + 杀进程重启
- 考虑开发时直接用 `uvicorn app.main:app` 不带 --reload，改完手动重启
